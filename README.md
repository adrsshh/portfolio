# Personal Portfolio

This repository contains the source code for my personal developer portfolio website. The portfolio is designed to present my skills, projects, and technical background in software development. It serves as a central place where recruiters, collaborators, and peers can learn about my work and explore the projects I have built.

The website follows a clean and responsive design approach so that it works smoothly across desktops, tablets, and mobile devices.

---

## Overview

The portfolio includes several sections that introduce my background, highlight my technical skills, and showcase selected development projects. The goal of this website is to provide a clear and organized representation of my work as a software developer.

---

## Features

* Responsive layout that adapts to different screen sizes
* Full-width section based layout for smooth scrolling navigation
* Dedicated sections for introduction, about, projects, and contact
* Organized project showcase with descriptions
* Simple and minimal user interface
* Optimized structure for easy updates and maintenance

## Running the Project Locally

To run this project on your local machine:

1. Clone the repository

```
git clone https://github.com/adrsshh/portfolio.git
```

2. Navigate to the project directory

```
cd portfolio
```

3. Open the main HTML file in your browser

```
index.html
```

No additional dependencies or installations are required.
